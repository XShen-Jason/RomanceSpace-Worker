表白小项目平台 — 技术架构与功能说明

一、项目目标

建立一个可分享、可定制的小项目平台，核心功能包括：

- 用户可领取专属三级域名，例如 user.example.com
- 用户可绑定现有模板的小项目（表白页面、倒计时、小游戏等）
- 用户可免费更换小项目模板或在模板允许范围内定制内容
- 平台可支持付费深度定制模板
- 高访问量下仍保持快速响应与稳定性

二、整体架构

平台采用 Cloudflare + 自建服务器 双层架构：

用户浏览器
│
▼
Cloudflare DNS
│
▼
Cloudflare Worker（边缘）
│  ├─ KV：存储子域名 → 模板映射
│  ├─ R2/CDN：存储图片、SVG、音频等资源
│  └─ 调用模板模块生成页面
│
▼
浏览器显示页面并执行前端互动 JS
│
▼
服务器 API（记录互动数据）
│
▼
数据库（用户信息、配置、互动记录）

核心原则：

- 访问快：用户访问走 Worker + KV + CDN
- 管理灵活：服务器处理注册、模板配置、用户数据、互动 API
- 高并发：Worker 天生支持水平扩展，热点页面可缓存
- 成本低：无需每个用户单独生成页面

三、Cloudflare 部分职责

1. DNS

  • 通配符解析 *.example.com
  • 支持用户动态领取三级域名，无需单独添加 DNS

2. Worker

核心任务：渲染小项目页面

1. 获取访问子域名（subdomain）
2. KV 查询 subdomain 对应模板
3. 读取模板模块
4. 获取用户配置（KV 缓存 / 服务器 API）
5. render(config) → HTML
6. 返回给用户浏览器

优化点：

  • 热门页面缓存到 Edge Cache
  • 模板模块动态 import，减少冷启动

3. KV（Key-Value）

  • 存储子域名与模板映射，例如： 
    {
  "xiaoming": "confession",
  "user2": "countdown"
}
  • 支持读多写少的高并发访问
  • 可缓存完整用户配置，减少服务器调用

4. R2 / CDN

  • 存储静态资源：图片、SVG、背景图、音乐等
  • 页面直接引用资源 URL，不增加 Worker 体积
  • 全球加速，响应快

四、服务器部分职责

1. 用户系统

  • 注册 / 登录
  • 用户信息管理
  • 子域名领取

2. 域名绑定 & 小项目配置

  • 用户领取三级域名：user.example.com
  • 绑定模板和用户配置： 
    - 配置 JSON 记录可定制字段（文本、背景、音乐等）
- 保存到数据库
- 更新 KV

3. 模板定制功能

  • 免费模板定制：用户可修改预定义字段
  • 付费深度定制：用户提交需求，平台制作新模板并绑定

4. 互动 API

  • 点赞、留言、点击统计等功能
  • Worker 页面前端 JS 调用服务器 API 记录数据
  • 数据存储在数据库中

六、用户使用流程（完整）

1. 注册账号
2. 领取三级域名：user.example.com
3. 选择模板小项目
4. 填写配置（可定制文本、图片、音乐等）
5. 分享链接
6. 朋友访问页面：
   Worker 获取域名 → KV 查询模板 → 读取配置 → render → 返回页面
7. 互动操作（点赞、留言）：
   前端 JS 调用 API，服务器记录数据
8. 用户更换模板或配置：
   更新数据库 → 更新 KV → Worker 下次访问自动生效

七、优化与扩展策略

1. Worker 模块化

  • 每个模板一个模块
  • 动态 import 减少冷启动
  • 单 Worker 放 30–60 个中等复杂项目可稳健运行
  • 超过 100 个项目 → 使用 Gateway Worker 拆分子 Worker

2. 缓存策略

  • 热门页面缓存到 Edge Cache
  • KV 缓存用户配置，减少服务器访问
  • 静态资源走 CDN / R2

3. 模板升级与深度定制

  • 普通模板修改字段 → 免费
  • 模板功能扩展 → 平台制作
  • 深度定制 → 新模板，付费

4. 互动与性能

  • Worker 渲染页面 → KV / R2
  • 互动 API → 服务器
  • 避免 Worker 内直接处理大量逻辑或数据库操作

八、总结

- 访问快：Cloudflare Worker + KV + CDN
- 管理灵活：服务器处理用户注册、模板配置和互动 API
- 成本低：模板 + 配置方式，避免为每个用户生成静态页面
- 可扩展：模块化模板设计，支持免费与付费定制
- 高并发：Worker 天生水平扩展，Edge Cache 缓存加速
核心理念：模板通用 + 配置可定制 + Worker 渲染 + 服务器管理 确保平台从几十用户到数十万用户都可平稳运行
